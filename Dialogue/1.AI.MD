  如何开始使用

  1. 安装依赖

  cd PersonalBlog
  npm install

  2. 启动开发服务器

  npm run dev

  访问 http://localhost:3000 查看你的博客

  3. 自定义内容

  编辑 src/data/posts.ts 文件来：
  - 修改个人信息（authorInfo 对象）
  - 添加、编辑或删除文章（posts 数组）

  4. 添加新文章

  在 posts 数组中添加新对象：

  {
    id: '6',
    title: '你的文章标题',
    excerpt: '文章简介',
    content: `# Markdown 格式的文章内容`,
    date: '2024-03-26',
    tags: ['标签1', '标签2'],
    author: '你的名字',
    readTime: 5
  }

  部署方案

  我已经在 DEPLOYMENT.md 文件中提供了详细的部署指南，支持：

  推荐方案（最简单）

  1. Vercel - 零配置，自动部署
  2. Netlify - 同样简单，自动 HTTPS

  其他方案

  3. GitHub Pages - 免费托管
  4. 传统服务器 - 使用 Nginx

  快速部署到 Vercel（3步搞定）

  # 1. 提交代码到 GitHub
  git init
  git add .
  git commit -m "Initial commit"
  git remote add origin https://github.com/你的用户名/你的仓库.git
  git push -u origin main

  # 2. 访问 vercel.com，用 GitHub 登录
  # 3. 导入你的仓库，点击 Deploy

  完成！你的博客就上线了，还有免费的 HTTPS 和 CDN。

  自定义建议

  1. 修改配色：编辑 tailwind.config.js 中的 colors.primary
  2. 添加评论：可以集成 Gitalk、Disqus 等评论系统
  3. 添加分析：集成 Google Analytics 或 Vercel Analytics
  4. SEO 优化：修改 index.html 中的 meta 标签
